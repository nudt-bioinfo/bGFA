class CLI::\+\+Option\+Base$<$ CRTP $>$}
This is the CRTP base class for \doxylink{class_c_l_i_1_1_option}{Option} and \doxylink{class_c_l_i_1_1_option_defaults}{Option\+Defaults}. It was designed this way to share parts of the class; an \doxylink{class_c_l_i_1_1_option_defaults}{Option\+Defaults} can copy to an \doxylink{class_c_l_i_1_1_option}{Option}.
