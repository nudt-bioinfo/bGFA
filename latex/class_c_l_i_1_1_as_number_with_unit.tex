\doxysubsection{Detailed Description}
Multiply a number by a factor using given mapping. Can be used to write transforms for SIZE or DURATION inputs.

Example:\+ With mapping = {\ttfamily \{"{}b"{}-\/\texorpdfstring{$>$}{>}1, "{}kb"{}-\/\texorpdfstring{$>$}{>}1024, "{}mb"{}-\/\texorpdfstring{$>$}{>}1024\texorpdfstring{$\ast$}{*}1024\}} one can recognize inputs like "{}100"{}, "{}12kb"{}, "{}100 MB"{}, that will be automatically transformed to 100, 14448, 104857600.

Output number type matches the type in the provided mapping. Therefore, if it is required to interpret real inputs like "{}0.\+42 s"{}, the mapping should be of a type \texorpdfstring{$<$}{<}string, float\texorpdfstring{$>$}{>} or \texorpdfstring{$<$}{<}string, double\texorpdfstring{$>$}{>}.
