\hypertarget{dir_f858fa0df565ccd1ee97085615bcf192_autotoc_md31}{}\doxysubsection{\texorpdfstring{WHAT IS THIS?}{WHAT IS THIS?}}\label{dir_f858fa0df565ccd1ee97085615bcf192_autotoc_md31}
The WFA2-\/lib offers tools that exploits the different libraries features and serve as examples on how to use and exploit the library functions. Also, these tools serve for testing the library and benchmarking its performance against other state-\/of-\/the-\/art tools and libraries for pairwise alignment. If you are interested in benchmarking WFA2-\/lib and other algorithms (internal or external), checkout the branch {\ttfamily benchmark}.

 1. GENERATE DATASET TOOL}{\label{_tool.generate}%
 1. GENERATE DATASET TOOL}}\label{dir_f858fa0df565ccd1ee97085615bcf192_autotoc_md32}
The {\itshape generate-\/dataset} tool allows generating synthetic random datasets for testing and benchmarking purposes. This tool produces a simple output format (i.\+e., each pair of sequences in 2 lines) containing the pairs of sequences to be aligned using the {\itshape align-\/benchmark} tool. For example, the following command generates a dataset named \textquotesingle{}sample.\+dataset.\+seq\textquotesingle{} of 5M pairs of 100 bases with an alignment error of 5\% (i.\+e., 5 mismatches, insertions, or deletions per alignment).


\begin{DoxyCode}{0}
\DoxyCodeLine{\$>\ ./bin/generate\_dataset\ -\/n\ 5000000\ -\/l\ 100\ -\/e\ 0.05\ -\/o\ sample.dataset.seq}

\end{DoxyCode}

The WFA2-\/lib includes the benchmarking tool {\itshape align-\/benchmark} to test and compare the performance of various pairwise alignment implementations. This tool takes as input a dataset containing pairs of sequences (i.\+e., pattern and text) to align. Patterns are preceded by the \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{} symbol and texts by the \textquotesingle{}\texorpdfstring{$<$}{<}\textquotesingle{} symbol. Example:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{>ATTGGAAAATAGGATTGGGGTTTGTTTATATTTGGGTTGAGGGATGTCCCACCTTCGTCGTCCTTACGTTTCCGGAAGGGAGTGGTTAGCTCGAAGCCCA}
\DoxyCodeLine{<GATTGGAAAATAGGATGGGGTTTGTTTATATTTGGGTTGAGGGATGTCCCACCTTGTCGTCCTTACGTTTCCGGAAGGGAGTGGTTGCTCGAAGCCCA}
\DoxyCodeLine{>CCGTAGAGTTAGACACTCGACCGTGGTGAATCCGCGACCACCGCTTTGACGGGCGCTCTACGGTATCCCGCGATTTGTGTACGTGAAGCAGTGATTAAAC}
\DoxyCodeLine{<CCTAGAGTTAGACACTCGACCGTGGTGAATCCGCGATCTACCGCTTTGACGGGCGCTCTACGGTATCCCGCGATTTGTGTACGTGAAGCGAGTGATTAAAC}
\DoxyCodeLine{[...]}

\end{DoxyCode}


Once you have the dataset ready, you can run the {\itshape align-\/benchmark} tool to benchmark the performance of a specific pairwise alignment method. For example, the WFA algorithm:\+
